    # 起步

    ubuntu密码：yjcoders
切换安装源：需要切换root用户
切换至root用户： sudo su
切换指定用户： su yjcoders

1、备份原来的源：sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup
2、编辑sources.list文件，更改为国内源：sudo vim /etc/apt/sources.list
  进入编辑器，将下面的阿里源，粘贴进去，替换原来的默认源地址
  deb http://mirrors.aliyun.com/ubuntu/ xenial main
  deb-src http://mirrors.aliyun.com/ubuntu/ xenial main

  deb http://mirrors.aliyun.com/ubuntu/ xenial-updates main
  deb-src http://mirrors.aliyun.com/ubuntu/ xenial-updates main

  deb http://mirrors.aliyun.com/ubuntu/ xenial universe
  deb-src http://mirrors.aliyun.com/ubuntu/ xenial universe
  deb http://mirrors.aliyun.com/ubuntu/ xenial-updates universe
  deb-src http://mirrors.aliyun.com/ubuntu/ xenial-updates universe

  deb http://mirrors.aliyun.com/ubuntu/ xenial-security main
  deb-src http://mirrors.aliyun.com/ubuntu/ xenial-security main
  deb http://mirrors.aliyun.com/ubuntu/ xenial-security universe
  deb-src http://mirrors.aliyun.com/ubuntu/ xenial-security universe

3、更新源：sudo apt-get update
4、更新软件：sudo apt-get upgrade

    # 切换时间

    timedatectl（查看时间状态）
timedatectl set-local-rtc 1 (将时钟调整为与本地时钟一致, 0 为设置为 UTC 时间)
timedatectl set-timezone Asia/Shanghai （设置系统时区为上海）

    # 1、基本操作

    进入编辑模式：i
退出编辑模式：esc
进入vim 编辑之后: 切换英文输入法状态，输入对应命令
退出  :q
强制退出  :q!
保存退出  :wq

### 撤销与恢复
撤销上一步：u
撤销在上一行的所有编辑操作： U
恢复：ctrl + r

### 移动
移动光标到行尾：shift + 4
移动光标到行首：shift + 6
移动光标到首行：gg
移动光标到尾行：G
移动到任意行：nG  (n是数字)

### 删除与粘贴
向后删除一个字符：x
向前删除一个字符：  X
删除/剪切光标所在行：dd
删除多行：3dd
删除/剪切光标到行尾所有字符：d$
复制光标所在行：yy
复制多行：3yy 会复制三行
复制光标到行尾所有字符： y$
向光标所在行下面粘贴：p
向光标所在行上面粘贴： P (大写)

    # 2.1、文件权限

    查看文件权限 ls -l
（r可读：4， w可写：2， x可执行：1， -没权限：0）
d rwx  rwx  rwx  = 777  表示目录的操作权限
- rwx  rwx  rwx  = 777  表示文件的操作权限
三组rwx 分别为文件所有者权限、用户组权限、其他用户权限

    # 3、虚拟机

    下载安装虚拟机：virtualBox:  https://www.virtualbox.org/wiki/Downloads
下载centOS系统：centOS:   http://isoredirect.centos.org/centos/8/isos/x86_64/CentOS-8-x86_64-1905-dvd1.iso
安装虚拟机，之后新建，点击存储，点击控制器，然后在右侧添加centOS系统盘，然后继续安装，安装之后，再回到存储位置，将此系统盘移除，否则会再次重新安装

进入虚拟机后，鼠标只能再虚拟机内部移动：
主机键（右侧的ctrl键）+ c 退出虚拟机
主机键 + Home键，弹出设置
