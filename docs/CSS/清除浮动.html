    BFC

    YJcoders 3/24/2018 CSS

    # 什么是 BFC?

    BFC(Box Formatting Context) 块级格式化上下文
是一个独立的容器，其中的元素布局和外面互不影响

    # BFC 特性

    BFC内相邻两个元素会发生margin边距重叠。
BFC内部元素和外部的浮动元素不会发生重叠。
BFC在计算高度时会把浮动元素计算进去。

    # 如何生成 BFC?

    1、float: left/right;
2、position: absolute/fixed;
3、overflow: hidden/auto/scroll;
4、display: flex/inline-block/table-cell

    # BFC 有什么作用？

    # 1、利用 BFC 解决 margin 重叠

    <div class="div">
  <div class="ch1"></div>
  <div class="ch2"></div>
</div>

<style>
  .ch1 {
    width: 100px;
    height: 150px;
    background: red;
    margin: 20px;
  }
  .ch2 {
    width: 100px;
    height: 250px;
    background: pink;
    margin: 20px;
  }
</style>

<!-- 两个盒子之间的margin只有20px，发生重叠 -->
<!-- 解决方式： -->
<div class="div">
  <div class="box"><div class="ch2"></div></div>
  <div class="ch2"></div>
</div>
<style>
  .box {
    overflow: hidden;
  }
</style>

    # 2、清除浮动，父盒子的高度没有被内容撑开，（定位和 float 引起）

    # 方案 1 ---- BFC 清除浮动

    1、给父元素设置overflow: auto/hidden/scroll
2、给父元素设置display: table
    1
    2

    # 方案 2 ---- clear 属性 - 直接添加

    对一个元素设置 clear 属性，那么该元素的上边框紧邻着浮动元素的 margin-bottom 渲染

    在浮动元素下方添加一个子盒子div 设置样式 clear: both
<div style="clear: both"></div>
    1
    2
    3

    # 方案 3 ---- 伪元素方式添加 clear 属性

    给浮动元素添加伪元素清除浮动样式

<div class="clearfix"></div>

.clearfix:after {
  content: '';
  display: block;
  clear: both;
}
